ls (list) - выводит список содержимого в папке (включая файлы и папки)

cs(chande directory) - переход между папками

cd ../- переход на одну папку назад

ctrl + l - очистить терминал

clear - очистить терминал

pwd (path workin directory) - абсолютный путь к папке где вы находитесь

ls -a  - показать содержимое папки (включая скрытые папки и файлы)

git init - ниницилизация git репозитория

git remote add origin ssh-url - привязка локального репозитория к удалённому

git remote -v - посмотреть к какому репозиторию привязан локальный файл

git remote remove (name) -удалить привязку
